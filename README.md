# 8 Puzzle using multiple search algorithm

## Search Types

1) Breadth First Search
2) Depth First Search
3) Iterative Deepening Search
4) Uniform Cost Search
5) Greedy Best First Search
6) A* Search

## Usage
- In main.cpp search type should be chosen(other ones should be commended out.)
- In include.hpp file there is a definition as PRINT_LOG which prints lots of logs
